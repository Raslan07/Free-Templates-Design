üìê Client Properties (clientLeft, clientTop, clientWidth, clientHeight)

Content Area

How Client Properties Work:

  • clientLeft/clientTop: Width of left/top border
  • clientWidth/clientHeight: Content area size (excluding borders & scrollbars)
clientWidth = element.width - leftBorder - rightBorder - scrollbarWidth
clientHeight = element.height - topBorder - bottomBorder - scrollbarHeight

üìè Offset Properties (offsetWidth, offsetHeight, offsetParent, offsetLeft, offsetTop)

Child

How Offset Properties Work:

  • offsetWidth/offsetHeight: Total element size including borders, padding, content
  • offsetLeft/offsetTop: Distance from offsetParent's padding edge
  • offsetParent: Nearest positioned ancestor or body
offsetWidth = content + padding + border
offsetLeft = distance from offsetParent's left padding edge

Scrollable Content

This content is larger than the container, so it creates scrollbars.

Scroll around and watch the properties update!

The scrollWidth and scrollHeight represent the total content size.

The scrollLeft and scrollTop show current scroll position.

How Scroll Properties Work:

  • scrollWidth/scrollHeight: Total content size including hidden overflow
  • scrollLeft/scrollTop: Current scroll offset from top-left corner
scrollWidth = total content width (visible + hidden)
scrollLeft = pixels scrolled horizontally from left edge
